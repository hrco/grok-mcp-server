# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-04

### Added
- Initial release of Grok MCP Server
- Chat completions with Grok models (grok-4, grok-4-1-fast, grok-3, grok-2)
- Web search with domain filtering and image understanding
- X/Twitter search with handle filtering, date ranges, and media understanding
- Session management for persistent conversation history
- Tool chaining for multi-step workflows:
  - Search and summarize
  - Multi-source research
  - Debug workflow
- Comprehensive documentation and examples
- MIT License

### Infrastructure
- Python 3.10+ support
- MCP protocol integration
- Persistent session storage
- Async/await architecture
